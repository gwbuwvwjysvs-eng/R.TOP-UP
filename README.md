# 🎮 R.TOP-UP - Free Fire Diamond Top-up Website

## 📋 Project Overview
**R.TOP-UP** is a complete Free Fire diamond top-up platform built with MERN Stack.

### 🎯 Complete Admin Panel Features:

#### 🏠 Dashboard
- মোট অর্ডার
- মোট আয়
- আজকের অর্ডার
- পেন্ডিং অর্ডার
- সম্পন্ন অর্ডার

#### 🎮 গেম ম্যানেজমেন্ট
- নতুন গেম যোগ
- গেম সম্পাদনা
- গেম মুছে ফেলা
- গেমের ছবি/লোগো আপলোড

#### 📦 প্যাকেজ ম্যানেজমেন্ট
- নতুন প্যাকেজ যোগ
- প্যাকেজ সম্পাদনা
- প্যাকেজ মুছে ফেলা
- ডায়মন্ড/UC/CP পরিমাণ পরিবর্তন
- দাম পরিবর্তন

#### 💳 পেমেন্ট সেটিংস
- বিকাশ নম্বর পরিবর্তন
- নগদ নম্বর পরিবর্তন
- রকেট নম্বর পরিবর্তন

#### 📋 অর্ডার ম্যানেজমেন্ট
- সব অর্ডার দেখা
- অর্ডার অনুমোদন (Approve)
- অর্ডার বাতিল (Reject)

#### 👥 ইউজার ম্যানেজমেন্ট
- সব ইউজারের তালিকা
- ইউজার ব্লক/আনব্লক

#### 🎟️ কুপন ম্যানেজমেন্ট
- কুপন তৈরি
- কুপন সম্পাদনা
- কুপন মুছে ফেলা

---

## 🛠️ Tech Stack

- **Frontend:** React.js + Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Authentication:** JWT
